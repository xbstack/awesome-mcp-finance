## 🧐 What is MCP in Finance?

The **Model Context Protocol (MCP)** is an open standard that enables AI models (like Claude 3.5 Sonnet, GPT-4o, or Cursor) to securely interact with local and remote financial data sources, Excel models, and trading APIs without custom integration for every agent.

In the finance world, MCP acts as the **standardized connector** between the LLM's reasoning and the hard logic of market data.

### 📊 Market Data
- [**mcp-server-yahoo-finance**](https://github.com/modelcontextprotocol/servers/tree/main/src/yfinance) - Official MCP server for real-time stock quotes, historical data, and financial statements via Yahoo Finance.

### 📈 Quant & Analysis
- [**mcp-quant-connect**](https://github.com/example/mcp-quant) - Connect your AI agent to QuantConnect for backtesting.
- [**mcp-excel-bridge**](https://github.com/example/mcp-excel) - Let AI read and write complex financial models directly in `.xlsx`.

### 📰 News & Sentiment
